## 💡 What is SafeCheck?
SafeCheck is a cybersecurity web app that detects 
phishing URLs and scam messages in real time.
Built specifically for Indian users to detect 
UPI fraud, KYC scams and phishing attacks.

## ⚡ Features
- 🔗 URL Scanner with risk score out of 100
- 💬 Message Scanner for WhatsApp/SMS scams
- 🇮🇳 Indian scam pattern detection (UPI, KYC, Paytm)
- 📊 Scan history
- 🛡️ 5 layer security analysis
